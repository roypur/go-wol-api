

to add a user

    ./wol-server add user <username> <password>

to delete a user

    ./wol-server delete user <username>
    
to edit a user

    ./wol-server edit user <username> <password>

to add a host

    ./wol-server add host <host> <mac>

to delete a host

    ./wol-server delete host <host>
    
to edit a host

    ./wol-server edit user <host> <mac>
 
to run server

    ./wol-server server
